# Análise Exploratória de Dados (EDA) - Transações Bancárias

## Descrição do Projeto
Este repositório contém uma Análise Exploratória de Dados (EDA) focada em um dataset de transações bancárias. O objetivo do projeto foi extrair *insights* sobre o comportamento dos clientes, identificar padrões de gastos, analisar distribuições e detectar anomalias que podem ser úteis para modelagem preditiva ou sistemas de detecção de fraude.

## Fonte dos Dados
Os dados utilizados nesta análise foram obtidos a partir do Kaggle: [Bank Transactions Data](https://www.kaggle.com/datasets/luisfernandolfss/bank-transactions-data-2).

## Ferramentas e Bibliotecas Utilizadas
* **Linguagem:** Python
* **Manipulação e Análise de Dados:** Pandas
* **Visualização de Dados:** Matplotlib, Seaborn
* **Ambiente de Desenvolvimento:** Google Colab

## Perguntas Respondidas na Análise
1. **Distribuição e Outliers:** Qual é a distribuição estatística do valor da transação e existem *outliers* significativos?
2. **Preferência de Canal:** Quais são os canais mais frequentes para cada tipo de transação?
3. **Idade vs. Saldo:** Existe correlação entre a idade do cliente e o saldo da conta?
4. **Impacto Geográfico:** Como o volume financeiro total varia por localização?
5. **Frequência de Uso:** Qual é o tempo médio entre a transação atual e a anterior por conta?
6. **Segurança e Valores:** Transações com muitas tentativas de login apresentam valores acima da média global?
7. **Ticket Médio por Profissão:** Quais ocupações possuem o maior ticket médio por transação?
8. **Duração vs. Canal:** Existe relação entre a duração da transação e o canal utilizado?
9. **Maiores Comerciantes:** Qual é o top 5 de comerciantes com maior volume financeiro transacionado?
10. **Comportamento Geracional:** Qual a proporção de tipos de transação por faixa etária?
